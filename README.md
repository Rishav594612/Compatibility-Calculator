# Compatibility-Calculator
Overview

The Friend Calculator is a browser-based application developed using HTML, CSS, and JavaScript. It calculates a compatibility percentage between two names and presents the result in a visually enhanced manner. The application incorporates animated UI elements, including a dynamic progress bar and floating heart effects, to provide an engaging user experience.

Key Features
Compatibility Calculation

The system computes a percentage value based on character codes and name lengths. The calculation is deterministic and intended strictly for entertainment purposes.

Animated Progress Bar

The resulting percentage is displayed with a smooth, visually appealing progress bar transition.

Floating Heart Animation

Upon calculation, animated heart elements appear within the UI to provide a celebratory effect.

Reset Function

A reset option clears user inputs, the displayed percentage, active animations, and restores the progress bar.

Responsive Layout

The interface is designed to adapt to various display sizes while maintaining clarity and ease of use.

Technology Stack

HTML5 for structure

CSS3 for styling and animations

JavaScript (Vanilla JS) for application logic and dynamic UI behavior

No external dependencies are required.

Usage Instructions

Download or copy the provided index.html file.

Open the file in any modern web browser.

Enter two names in the input fields.

Select Calculate Love % to generate the result.

Use Reset to clear all fields and animations.
